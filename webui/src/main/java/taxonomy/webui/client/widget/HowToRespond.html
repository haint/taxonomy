HOW TO RESPOND (DEPRECATED)

You're already known how to respond to the system !
Just follow the prompt, then type your choice at this sign: ===>. Use any combination of uppercase/lowercase (although uppercase is ussually preset).
If you do not use all of the space provided at the prompt, press ENTER when you finish your answer.
If the provided space already contains data (default), you may:

  • either press ENTER to accept the default data
  • or type new data, ingoring old data (no need to delete it)

Some control may be in effect, eg. you can't enter letters in numeric fields. In this case, what you enter will not appear; just try again.
The system will repeat the same prompt, if you choose an incorrect answer. Otherwise, it will not know how to proceed!
In multi-field prompts, to go to the next field use key ENTER, and go to the previous field use key ARROW UP
