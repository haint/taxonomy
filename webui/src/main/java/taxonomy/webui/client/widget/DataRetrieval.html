HOW TO WORK WITH DATA RETRIEVAL (Will be replaced by FullText search feature)

Just start experiment and enjoy it! You may want to try Keyword Listing to list terms currently used in this database, but this is seldom necessary.

Enter a word in field provided, and the system will search for records with the corresponding fields containing that word anywhere in the name.

For example, with the criteria of: English name of CARP, you will get records with English names like 'CARP ,COMMON' and 'CARP,GRASS', meaning all carps in the database.

To search for compounds terms, entire words at left or right can be omitted: 'DEVELOPMENT' will give both 'so-and-so development' and 'development so-and-so'. But words can be right-truncated only. 'CHEMI' in the search criteria will give CHEMISTRY, CHEMICAL ENGINEERING..., but not BIOCHEMISTRY.

Leave blank for any unused retrieval criterion, and this means any.

Retrieved information displayed may contain data from more than one databases and not the same as Record Listing display which works with single databases.
